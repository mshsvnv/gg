\chapter{Сравнение методов}

Сравнение проводится по двум группам показателей: вычислительной эффективности (FLOPs, число параметров, время инференса) и качеству синтеза (PSNR, SSIM, LPIPS, DISTS).

\section{Экспериментальная установка}

\subsection*{Модели для сравнения}

В рамках исследования обучены и протестированы следующие модели на основе архитектуры IFRNet:
\begin{itemize}
	\item \texttt{IFRNet (baseline)}~---~оригинальная архитектура, предложенная в работе~\cite{IFRNet2022}. Использует полное цветовое пространство RGB и стандартную функцию потерь~---~комбинацию L1, Ternary и дистилляционного лосса;

	\item \texttt{IFRNet\_RGB2}~---~модификация с равномерным сокращением числа каналов в два раза на всех уровнях пирамиды признаков (16, 24, 36, 48 вместо 32, 48, 72, 96). Функция потерь идентична базовой модели;

	\item \texttt{IFRNet\_YUV420}~---~модификация с переходом к цветовому пространству YUV420 и интерполяцией на основе яркостной компоненты $Y$. Цветоразностные компоненты $U$ и $V$ переносятся с использованием оптического потока, вычисленного по яркости. Функция потерь идентична базовой модели;

	\item \texttt{IFRNet\_RGB2+LPIPS}~---~архитектура \texttt{IFRNet\_RGB2} с добавлением перцептуальной функции потерь LPIPS~\cite{zhang2018} в состав общей функции потерь.
\end{itemize}

\subsection*{Показатели оценки}

Оценка выполнялась по двум группам показателей.

Показатели вычислительной эффективности:
\begin{itemize}
	\item \textbf{FLOPs} (Floating Point Operations)~---~количество операций с плавающей запятой, необходимое для обработки одного кадра. Рассчитывалось с помощью библиотеки \texttt{thop} для изображений размером $1920 \times 1088$;
	\item \textbf{Params}~---~число обучаемых параметров модели (в миллионах);
	\item \textbf{Inference Time}~---~среднее время инференса на один кадр (в секундах).
\end{itemize}

Показатели качества:
\begin{itemize}
	\item \textbf{PSNR} (Peak Signal-to-Noise Ratio)~---~пиксельная метрика, характеризующая отношение сигнал/шум в децибелах;
	\item \textbf{SSIM} (Structural Similarity Index)~---~метрика структурного сходства, учитывающая яркость, контраст и структуру изображения~\cite{wang2004};
	\item \textbf{LPIPS} (Learned Perceptual Image Patch Similarity)~---~перцептуальная метрика, использующая признаки предобученной сети VGG для оценки визуального сходства~\cite{zhang2018};
	\item \textbf{DISTS} (Deep Image Structure and Texture Similarity)~---~перцептуальная метрика, раздельно оценивающая структурное и текстурное сходство изображений~\cite{ding2020}.
\end{itemize}

\subsection*{Набор данных и условия обучения}

Обучение всех моделей проводилось на наборе данных Vimeo90K. Входные изображения имели разрешение $1920 \times 1088$ пикселей. Для модели с компонентой LPIPS в функции потерь использовался весовой коэффициент $\lambda = 0{,}1$.

\subsection*{Анализ эффективности}

\textbf{Сокращение вычислительной сложности.} Все предложенные методы оптимизации обеспечивают снижение вычислительной сложности по сравнению с базовой моделью IFRNet. За счёт сокращения ширины сети модели \texttt{IFRNet\_RGB2} и \texttt{IFRNet\_RGB2+LPIPS} показывают снижение FLOPs в 3,8 раза (с 479,441G до 126,191G) и ускорение инференса более чем в два раза (с 0,032 до 0,015 с на кадр).

Модель \texttt{IFRNet\_YUV420} демонстрирует снижение FLOPs до 122,682G и время инференса 0,014 с/кадр, что несколько ниже, чем у RGB2-версий, за счёт обработки цветоразностных компонент на пониженном разрешении.

\textbf{Число параметров.} Сокращение ширины сети в два раза позволило уменьшить число параметров с 4,96M до 1,00M. Модель на основе YUV420 сохраняет это число параметров, поскольку архитектура сети не изменяется (изменения касаются только входных данных и стратегии синтеза цветовых компонент).

\subsection*{Анализ качества}

\textbf{Пиксельные метрики.} Все оптимизированные модели показывают снижение PSNR и SSIM относительно базовой архитектуры. Снижение PSNR составляет 0,7--0,9 дБ (с 36,053 дБ до 35,1--35,3 дБ), снижение SSIM не превышает 0,004, что является приемлемой ценой за достигнутое ускорение. Модель \texttt{IFRNet\_YUV420} незначительно уступает \texttt{IFRNet\_RGB2} по пиксельным метрикам (PSNR 35,239 дБ против 35,340 дБ).

\textbf{Перцептуальное качество.} Наиболее существенный эффект достигается при добавлении перцептуальной функции потерь LPIPS в процесс обучения. Модель \texttt{IFRNet\_RGB2+LPIPS} показывает LPIPS = 0,0361 и DISTS = 0,0222, что ниже не только соответствующих значений модели \texttt{IFRNet\_RGB2} (0,0524 и 0,0362), но и значений базовой модели (0,0503 и 0,0342). Таким образом, при сохранении вычислительной эффективности на уровне \texttt{IFRNet\_RGB2} модель \texttt{IFRNet\_RGB2+LPIPS} превосходит базовую архитектуру по перцептуальным показателям.

Модели, обученные без компоненты LPIPS (\texttt{IFRNet\_RGB2} и \texttt{IFRNet\_YUV420}), уступают базовой модели по перцептуальным метрикам: переход к YUV420 приводит к наибольшему росту LPIPS (0,0557) и DISTS (0,0416), что связано с обработкой цветоразностных компонент на пониженном разрешении.

По соотношению качество/эффективность рассмотренные модели можно разделить на три группы:

\begin{enumerate}
	\item \textbf{наилучший баланс}~---~\texttt{IFRNet\_RGB2+LPIPS} обеспечивает снижение FLOPs в 3,8 раза и ускорение инференса более чем в два раза при наилучших перцептуальных показателях среди всех моделей (LPIPS = 0,0361, DISTS = 0,0222), превосходящих в том числе базовую архитектуру;

	\item \textbf{максимальная эффективность}~---~\texttt{IFRNet\_RGB2} и \texttt{IFRNet\_YUV420} обеспечивают наибольшее снижение вычислительных затрат (в 3,8--3,9 раза по FLOPs) при умеренном снижении качества; вариант YUV420 несколько быстрее, но уступает по перцептуальным метрикам;

	\item \textbf{базовая модель}~---~\texttt{IFRNet (baseline)} обеспечивает наилучшие пиксельные метрики (PSNR, SSIM) ценой существенно более высоких вычислительных затрат.
\end{enumerate}
